Como usar o Brother's Businesses

1. Cadastre-se

Clique em "Entrar / Cadastrar" no topo e informe seu nome.

2. Anuncie

Vá para a aba "Quero Vender", preencha os dados e publique.

3. Pesquise

Na aba "Quero Comprar", use a busca para encontrar produtos.

4. Gerencie

Você só pode excluir os anúncios que criou.

5. Contato

Utilize o formulário de WhatsApp para negociar ou falar com a equipe.

Todos os dados são salvos automaticamente no seu navegador (localStorage).
